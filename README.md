# firebase_flutter

Projeto feito após estudos de como realizar uma autenticação de Login utilizando Flutter + firebase

Para rodar:

- Abrir um cmd

Rodar os comandos:

dart pub global activate flutterfire_cli
firebase login
flutterfire configure --project=brademoproj1


Ao rodar o comando abaixo, o projeto já pode ser desenvolvido

flutter pub add firebase_auth cloud_firestore firebase_storage
